# Notas del curso de C/C++

- Primer y segundo día han sido de configuración de Vscode, Git y el compilador C/C++
- Tercer día: Repaso de conceptos básicos de programación en C/C++: variables, tipos de datos, estructuras de control, funciones, punteros, etc.
- Cuarto día: Ejercicios prácticos para afianzar los conceptos aprendidos, incluyendo la implementación de algoritmos básicos como ordenamiento y búsqueda.
- Quinto día: Introducción a la programación orientada a objetos en C++, con ejemplos de clases, objetos, herencia y polimorfismo.
- Sexto día: Ejercicios avanzados que combinan conceptos de programación estructurada y orientada a objetos, como la creación de una pequeña aplicación de gestión de datos.
